About Wispy Core

Wispy Core is a modern, multisite content management system built with Go. It's designed to be fast, secure, and developer-friendly while providing a beautiful user experience.

Key Features

  • Multisite Architecture: Manage multiple websites from a single installation
  • High Performance: Built with Go for exceptional speed and efficiency
  • Modern Templates: Powered by daisyUI and Tailwind CSS
  • Flexible Content: HTML-based page structure with template inheritance
  • Developer Experience: Clean architecture and extensible design

Getting Started

To get started with Wispy Core, check out our documentation or visit the admin panel to create your first pages.
