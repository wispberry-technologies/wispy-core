Caching Performance Demonstration

Multiple identical requests to demonstrate caching performance:

{{ getAPIData $response "_duration_ms" }}ms

Note: The first request should be slower and marked as "Fresh", while subsequent requests should be much faster and marked as "Cached".

Template Function Reference

Basic API Call

{{ `{{ $response := api "GET" "/health" }}
{{ $response.status_code }}` }}

API Call with Headers

{{ `{{ $response := api "GET" "/api/v1/data" (dict "headers" (dict "Authorization" "Bearer token")) }}` }}

POST Request with Body

{{ `{{ $response := api "POST" "/api/v1/create" (dict 
    "headers" (dict "Content-Type" "application/json")
    "body" (dict "name" "New Item" "type" "example")
) }}` }}

Error Handling

{{ `{{ $response := api "GET" "/api/v1/data" }}
{{ if isAPIError $response }}
    Error: {{ getAPIError $response }}
{{ else }}
    Success: {{ getAPIData $response "result" }}
{{ end }}` }}
